# YT-DLP-Interface
Custom additional functionality added to yt-dlp written in python

- cli for Downloading YT videos via YT-DLP
- conversion of videos to mp3
- fetch data from MusicBrainz to identify tracks --> edit --> store to DB (WIP)
- editing of id3 tags of downloaded files 
- - file naming convention accoring to JellyFin + DBID (WIP)
- Tracking downloaded music in mysql database (WIP)
- Organising Tracks according to JellyFin rules (WIP)
